## Калькулятор

Это простой калькулятор, написанный с помощью React JS, который умеет складывать и вычитать.

## Инструкция

1. Склонируйте репозиторий
```shell
git clone https://github.com/powder-rose/Calculator.git
```
2. Установите зависимости
```shell
npm i
```
3. Готово! Можете запустить с помощью дев сервера.
##

Запустить дев сервер:

```shell
npm run start
```
Сборка:

```shell
npm run build
```
